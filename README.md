# EDA-
# 📊 Customer Churn Analysis (EDA + Data Visualization)

## 🚀 Project Overview
This project presents a complete **Exploratory Data Analysis (EDA)** and **Data Visualization** on a large-scale customer dataset containing over **100,000 records**. The goal is to analyze customer behavior and identify key factors influencing customer churn.

---

## 🎯 Objectives
- Understand dataset structure and features  
- Identify trends, patterns, and anomalies  
- Analyze factors affecting customer churn  
- Create meaningful visualizations for insights  

---

## 📁 Dataset
The dataset includes the following features:
- CustomerID  
- Age  
- Gender  
- Tenure  
- MonthlyCharges  
- ContractType  
- InternetService  
- PaymentMethod  
- City  
- Churn  

---

## 🧪 Task 2: Exploratory Data Analysis (EDA)
- Data loading and inspection  
- Data type verification  
- Missing value check  
- Statistical summary  
- Group-based analysis (Churn vs Features)  
- Outlier detection  
- Correlation analysis  

### 🔍 Key Insights
- High monthly charges increase churn probability  
- Customers with low tenure churn more  
- Monthly contract users show higher churn  
- Long-term contracts improve retention  

---

## 📊 Task 3: Data Visualization
Visualization techniques used:
- Count Plots  
- Box Plots  
- Heatmaps  
- Bar Charts  
- Pair Plots  

### 📈 Findings
- Strong relationship between **Monthly Charges and Churn**  
- Tenure plays a critical role in retention  
- Contract type significantly impacts churn  
- Regional and payment patterns show variation
